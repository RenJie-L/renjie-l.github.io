浙江省新苗人才计划科研项目。相关成果发表于 _Deep Reinforcement Learning-based Swing-Free Trajectories Planning Algorithm for UAV with a Suspended Load_（China Automation Congress 2022）。

> 论文研究如何让四旋翼无人机通过绳索运输吊载时，在快速到达目标点的同时抑制负载摆动。核心创新是设计了一种**分区域渐近式奖励函数**，让智能体在远离目标点时优先快速接近目标，在靠近目标点后逐步提高对吊载摆角的约束，最终实现较快、较稳的吊运过程。

## 一句话概括

论文将无人机吊载运输建模为一个二维欠驱动系统，并使用 **Deep Q-Network（DQN）** 学习飞行轨迹；核心创新是设计了一种 **分区域渐近式奖励函数**，让智能体在远离目标点时优先快速接近目标，在靠近目标点后逐步提高对吊载摆角的约束，最终实现较快、较稳的吊运过程。

## 1. 研究背景

无人机可以用于食品、药品、应急物资等快速运输。当货物形状不规则，或无法直接固定在机体下方时，常采用绳索吊运。

但吊载系统具有三个主要难点：

- **欠驱动**：负载摆角不能被直接控制，只能通过无人机运动间接调节；
- **强耦合**：无人机位置、速度、姿态与吊载摆动相互影响；
- **非线性**：快速加速或减速容易引起明显摆动，增加飞行不稳定性。

因此，轨迹规划不能只追求“尽快到达”，还必须同时考虑负载摆角和摆动速度。

## 2. 论文解决的问题

论文希望生成一条满足以下目标的轨迹：

1. 无人机从起点快速飞到目标点；
2. 飞行过程中吊载摆角不超过允许范围；
3. 到达目标点后，无人机速度和吊载摆角都趋近于零；
4. 强化学习训练能够稳定收敛，并减少无效探索。

## 3. 系统建模

论文假设：

- 无人机机架为均匀对称刚体；
- 绳索连接点与无人机质心重合；
- 绳索始终绷紧，且不发生弹性形变；
- 无人机姿态角被限制在实际可飞行范围内。

由于吊载在两个垂直平面内的摆动可以近似解耦，论文将三维问题简化为二维平面运动。系统状态主要包含：

- 无人机在两个方向的位置与速度；
- 吊索摆角与摆角速度；
- 无人机总推力和俯仰角等控制输入。

这种降维处理降低了强化学习状态空间的复杂度，也提升了训练速度。

## 4. DQN 轨迹规划方法

DQN 使用神经网络估计状态-动作价值函数：

$$
Q^*(s,a) \leftarrow Q(s,a)+\alpha\left[r+\gamma\max_{a'}Q(s',a')-Q(s,a)\right]
$$

算法包含：

- **主网络（Main Network）**：根据当前状态输出各离散动作的 Q 值；
- **目标网络（Target Network）**：计算相对稳定的目标 Q 值；
- **经验回放（Experience Replay）**：随机采样历史交互数据，减弱样本相关性；
- **$\epsilon$-greedy 策略**：在探索新动作和利用已学策略之间平衡。

## 5. 核心创新：分区域渐近式奖励函数

单纯使用距离奖励会导致无人机为了尽快接近目标点而突然加速，从而产生较大摆动；单纯强调摆角，又可能使无人机在目标点附近悬停，无法准确到达。

论文将奖励拆分为两部分。

### 距离奖励

$$
r_d = d_{start,target}-d_{current,target}
$$

当无人机逐渐接近目标点时，距离奖励增大。

### 摆角奖励

$$
r_\theta = \pi-|\theta-\theta_T|
$$

吊载摆角越接近期望角度，奖励越高。

### 加权组合

$$
r_{Area}=\lambda r_d+\delta r_\theta
$$

其中：

- $\lambda$ 控制对位置误差的重视程度；
- $\delta$ 控制对吊载摆角的重视程度。

论文进一步将目标附近划分为多个逐渐缩小的区域：

- **Area 1**：主要鼓励无人机接近目标点；
- **Area 2 / Area 3**：同时考虑距离和摆角，且逐渐增强摆角约束；
- **Area 4**：无人机已基本到达目标点，重点抑制剩余摆动，使摆角趋近于零。

如果无人机偏离规定区域或摆角超出允许范围，则给予负奖励。

这一设计相当于把复杂任务拆成“先接近、再减速、最后消摆”三个阶段，使奖励信号更符合实际吊运过程。

## 6. 仿真环境与参数

论文基于以下工具搭建训练环境：

- Python 3.8
- PyTorch
- OpenAI Gym
- 自主开发的无人机吊载仿真环境

主要训练参数如下：

| 参数 | 数值 |
| --- | ---: |
| Batch Size | 64 |
| Replay Buffer Size | 100000 |
| 折扣因子 $\gamma$ | 0.99 |
| 学习率 $\alpha$ | 0.001 |
| 初始探索率 $\epsilon$ | 0.99 |
| 最大步数 | 330 |
| 训练轮数 | 850 |
| 隐藏层维度 | 256 |

仿真区域大小为 $4\times4$，目标位置约为 $(2\text{ m},1\text{ m})$。训练完成后进行了 20 组测试。

## 7. 实验结果

### 7.1 位置与速度

无人机在起始阶段平稳加速，约 2 秒接近目标点，随后减速。轨迹会轻微越过目标点，再回到目标位置，最终两个方向的速度都收敛到零。

### 7.2 吊载摆动

实验中：

- 最大摆角控制在约 **16°** 以内；
- 最大摆角速度控制在约 **46°/s** 以内；
- 接近目标点后，摆角逐渐减小并最终在零附近稳定。

### 7.3 典型飞行轨迹

典型轨迹从 $(0,0)$ 出发，经由约 $(0.56,0.51)$、$(1.27,0.86)$ 和 $(2.14,1.06)$，最终稳定在约 $(1.97,1.04)$。

这说明策略会主动形成一个“加速—减速—微调”的过程，而不是直接冲向目标点。

### 7.4 收敛性

累计奖励在训练前期波动较大，约在 **600 个 episode** 后趋于稳定。在此之后，无人机在大多数测试中都能持续、准确地到达目标点。

## 8. 与已有方法的比较

论文将结果与已有基于强化学习和概率路线图的方法进行对比：

- 已有方法的摆角变化更平缓；
- 本文方法在起始阶段可能产生略大的摆角；
- 但本文方法到达目标点所需时间更短，奖励收敛更快；
- 最终摆角同样能够收敛至零附近。

因此，该方法在运输效率与防摆稳定性之间取得了较好的平衡。

## 9. 论文主要贡献

1. **建立了适用于强化学习的无人机吊载二维动力学环境**，并实现与 OpenAI Gym 兼容的仿真平台；
2. **将连续控制过程离散化为 DQN 动作空间**，降低训练难度并提升训练速度；
3. **提出分区域渐近式奖励函数**，动态协调“接近目标”和“抑制摆动”两个目标；
4. **通过仿真验证了方法有效性**，无人机能够快速到达目标点，同时将吊载摆动控制在限定范围内。

## 10. 局限性与后续方向

论文当前主要基于二维模型和仿真环境，动作空间也经过离散化。后续工作包括：

- 使用 PPO、A3C 等适合连续控制的深度强化学习算法；
- 将动作空间改为连续空间，使推力与姿态变化更加平滑；
- 扩展到完整三维吊载运动；
- 加入风扰动、模型误差、传感器噪声和障碍物；
- 将训练策略部署到真实无人机平台进行实验验证。

## 结论

这篇论文的本质，是通过重新设计强化学习奖励机制，引导无人机学会一套符合吊运物理规律的运动策略：远处快速接近目标，近处主动减速，并在到达后消除剩余摆动。

仿真结果表明，分区域奖励函数能够明显改善 DQN 在多目标任务中的训练效果，使无人机在较短时间内完成吊载运输，并把最大摆角控制在约 16° 以内。该工作为强化学习在无人机欠驱动吊载系统中的应用提供了一个可行框架。
