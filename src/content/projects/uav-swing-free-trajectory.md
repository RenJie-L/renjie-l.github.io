## 项目概览

这是一个浙江省新苗人才计划科研项目。研究面向绳索悬挂负载运输：无人机既要快速、准确地抵达目标点，也要抑制加减速引发的载荷摆动。项目基于简化后的二维吊载动力学构建 OpenAI Gym 仿真环境，并使用 Deep Q-Network（DQN）学习兼顾运输效率与摆动抑制的飞行轨迹。

> 本论文完成的是动力学仿真与算法验证，未进行真实无人机部署。真实环境部署在论文中被列为后续研究方向。

## 研究问题

当负载通过绳索悬挂在四旋翼下方时，无人机与负载共同组成一个强耦合、非线性、欠驱动系统。无人机的加速、减速和方向变化都会使负载产生类似摆锤的振荡；过大的摆角不仅降低运输效率，也会影响飞行稳定性。

论文把任务定义为一个多目标轨迹规划问题：

- 在有限时间内从起点到达指定目标位置；
- 限制运输过程中的负载摆角和角速度；
- 到达目标点时，使无人机速度和负载摆动趋近于零；
- 在“快速到达”与“平稳运输”之间取得平衡。

## 系统建模

### 二维平面简化

论文假设无人机为均匀对称刚体，绳索悬挂点与无人机质心重合，并且绳索始终绷紧、无弹性形变。由于负载在两个垂直平面内的摆动可以解耦，三维运动被拆分为两个相互独立的二维平面问题，从而降低强化学习的状态维度和训练难度。

单个平面内使用六维可观测状态：

```text
s = [y, ẏ, z, ż, α, α̇]
```

其中 `y、z` 表示无人机位置，`ẏ、ż` 表示对应速度，`α` 为悬挂绳摆角，`α̇` 为摆动角速度。系统输入由无人机总推力 `U` 和俯仰角 `θ` 构成；已知参数包括无人机质量 `M`、负载质量 `m` 与绳长 `l`。

### 仿真环境

基于上述动力学模型，项目开发了兼容 OpenAI Gym 的训练环境。动作空间经过离散化，使 DQN 可以直接输出每个候选动作的 Q 值，同时减少连续控制带来的探索成本。环境能够实时返回位置、速度、负载摆角和奖励，并提供图形化训练结果。

## DQN 轨迹规划方法

DQN 使用神经网络近似状态—动作价值函数 `Q(s, a)`。智能体通过 ε-greedy 策略选择动作，并将每一步的 `(s, a, r, s')` 存入经验回放池；训练时随机采样小批量数据，降低连续样本之间的相关性。

算法同时维护两个结构相同的网络：

- **主网络（Main Net）**：输出当前状态下各动作的 Q 值并持续更新；
- **目标网络（Target Net）**：计算相对稳定的目标 Q 值，每隔一定步数从主网络同步参数。

Q-learning 的目标值与均方误差损失可概括为：

```text
TargetQ = r + γ · max Q(s', a')
Loss = E[(TargetQ - Q(s, a))²]
```

## 分区渐近探索奖励

普通距离奖励容易让无人机为了尽快抵达目标而突然加速，导致负载大幅摆动；如果摆角奖励权重过高，无人机又可能停在目标区域之外。论文因此设计了分区渐近探索奖励函数，同时考虑目标距离与摆角：

```text
r_area = λ · r_distance + δ · r_angle
```

其中 `λ` 控制位置误差的重要性，`δ` 控制摆动抑制的重要性。目标周围被划分为多个逐步缩小的区域，奖励权重随无人机所在区域动态调整：

1. 在较远区域，以距离奖励为主，鼓励无人机有效接近目标；
2. 进入中间区域后，同时提高距离与摆角奖励，并逐步增加摆角权重；
3. 到达最内层区域时，进一步强调摆动抑制，使无人机减速并稳定停留；
4. 未进入有效探索区域或摆角超限时给予负奖励，引导策略回到可行状态。

这种奖励塑形最终形成了“快速接近—主动减速—轻微越过—回调稳定”的轨迹特征。

## 训练配置

| 参数           |       设置 |
| -------------- | ---------: |
| 仿真区域       |  4 m × 4 m |
| 目标点         | (2 m, 1 m) |
| Batch size     |         64 |
| 经验回放池     |    100,000 |
| 折扣因子 γ     |       0.99 |
| 学习率 α       |      0.001 |
| 初始 ε         |       0.99 |
| 每回合最大步数 |        330 |
| 训练回合       |        850 |
| 隐藏层维度     |        256 |

仿真使用 Python 3.8、PyTorch GPU 与 OpenAI Gym，论文记录的训练设备为 AMD R5-3600、16 GB 内存和 RTX 2060。

## 实验结果

训练完成后，论文对策略进行了 20 组仿真测试。无人机在起始约 1 秒内逐渐加速，约 2 秒时接近目标点并开始减速；在轻微越过目标后回调，最终位置约为 `(1.97 m, 1.04 m)`，两个方向的速度均收敛到零附近。

主要结果包括：

- 整个运输过程的最大负载摆角控制在 **16°** 以内；
- 最大摆动角速度控制在 **46°/s** 以内；
- 累计奖励在约第 **600** 个训练回合后趋于稳定；
- 训练至 **850** 个回合时，智能体在多数测试中能够稳定、准确地抵达目标点；
- 与论文引用的 PRMs 方法相比，该方法在仿真中表现出更快的收敛和更短的到达时间，但起始阶段摆角相对更大。

## 贡献与局限

项目的主要贡献是把吊载无人机的动力学模型、Gym 训练环境、DQN 与分区奖励塑形组合成一套完整的轨迹生成流程。分区奖励避免了单一距离奖励与单一抑摆奖励之间的冲突，使策略能够根据接近目标的程度改变控制重点。

论文仍有以下局限：

- 动作空间经过离散化，生成轨迹的平滑程度受限；
- 结果来自简化二维模型和仿真环境，尚未验证真实飞行中的风扰、传感噪声与模型误差；
- 论文没有覆盖复杂障碍物、三维耦合运动或变化负载等情况。

后续方向包括使用 A3C、PPO 等支持连续动作空间的算法获得更平滑的控制结果，并将训练策略部署到真实无人机平台进行验证。
